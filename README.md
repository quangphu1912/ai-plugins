# ai-plugins

Claude Code plugin marketplace by Phu Le.

## Plugins

### codebase-analyzer

Progressive-depth codebase analysis, reverse engineering, and development orchestration for Claude Code, OpenCode, and Codex.

- 32 skills (20 analysis + 12 orchestration)
- 3 agents (code-explorer, behavior-simulator, code-reviewer)
- Multi-platform support with honest degradation

## Install

```bash
/plugin marketplace add quangphu1912/ai-plugins
/plugin install codebase-analyzer@ai-plugins
```

## Adding a New Plugin

Add an entry to `marketplace.json` with the plugin name, description, version, and GitHub source.
